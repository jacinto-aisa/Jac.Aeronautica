# Jac.Aeronautica
EjemploPasoAPaso

Proyecto en el que se como comunicar varios servicios o microservicios mediante eventos y mensajes.
También se aplican los patrones del circuit breaker y replay en las factorias de contextos.
Estan también los proyectos de prueba por lo que se puede desplegar el azure y dentro de una canalización.
Tiene también la implementación del patrón specification y puede servir para entender también la configuración de la aplicación, 
ya sea por slots, con Azure configuration.
El proyecto no está en contenedores, pero puede ser usado para pruebas
